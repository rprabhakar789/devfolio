# Devfolio - Interactive Portfolio

An interactive, modern portfolio website built with React and Framer Motion.

## Features

✨ **Interactive Design**
- Smooth animations and transitions
- Gradient backgrounds and glassmorphism effects
- Responsive grid layouts
- Scroll animations

🎯 **Complete Sections**
- Hero section with code visualization
- About section with stats
- Timeline-based experience showcase
- Skills filter and progress bars
- Featured projects gallery
- Contact form and social links
- Footer with creative touches

🚀 **Performance**
- Built with Vite (instant HMR)
- Optimized bundle size (~105KB gzipped)
- Lazy-loaded components

📱 **Responsive**
- Mobile-first design
- Tablet optimized
- Desktop-enhanced animations

## Tech Stack

- **React 19** - UI Library
- **Framer Motion** - Animation library
- **Vite** - Build tool
- **React Icons** - Icon library
- **CSS3** - Modern styling with gradients and animations

## Quick Start

```bash
npm install
npm run dev      # Development server
npm run build    # Production build
```

## GitHub Pages Deployment

Update `vite.config.js` with your username:
```js
base: '/devfolio/',
```

Push to GitHub and enable Pages in Settings → Pages.

Or use GitHub Actions with the deploy workflow.

## Customization

Edit CSS variables in `src/App.css` for colors and theming.
Update component content with your own data.
